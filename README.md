# 掃除のさとう Release 1.0

GitHub Pages 公開用の完成データです。

## ファイル構成

- `index.html`
- `style.css`
- `script.js`
- `assets/` 画像・ロゴ一式

## 公開手順

1. GitHubで新規リポジトリを作成
2. このフォルダ内のファイルをリポジトリ直下へアップロード
3. Settings → Pages → Branch を `main` / `/root` に設定
4. 表示されたURLにアクセス
